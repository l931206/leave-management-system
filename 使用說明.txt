建議更新方式：
1. 先備份目前專案。
2. 解壓縮這一包。
3. 建議直接以本包資料夾作為新版專案，不要整包硬塞進舊資料夾。
4. 若要保留舊的 Git 紀錄，可把本包內容複製到原專案根目錄，但不要複製外層資料夾。
5. 建立虛擬環境並安裝：
   python -m venv .venv
   .\.venv\Scripts\python.exe -m pip install -r requirements.txt
6. 執行：
   .\.venv\Scripts\python.exe manage.py check
   .\.venv\Scripts\python.exe manage.py migrate
   .\.venv\Scripts\python.exe manage.py runserver
7. 測試主管帳號：
   manager / demo1234
8. 確認後提交 Git：
   git add .
   git commit -m "Sprint 5: add manager management pages and clean project"
   git push origin main
