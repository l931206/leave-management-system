# 專案結構說明

## 根目錄

根目錄只保留執行、部署及專案說明所需的檔案。

| 路徑 | 用途 |
|---|---|
| `config/` | Django 設定、URL 入口與 WSGI |
| `leaveapp/` | Models、Views、Forms、URLs、管理指令 |
| `templates/` | 登入、員工、主管頁面 |
| `static/` | CSS 與 JavaScript |
| `media/` | 上傳附件 |
| `docs/` | 規格、資料庫、部署與測試文件 |
| `scripts/` | Windows、macOS、Linux 快速啟動腳本 |
| `screenshots/` | GitHub 或報告使用的系統截圖 |

## 已移除的內容

打包版本不包含：

- `.git/`
- `.venv/`
- `staticfiles/`
- `__pycache__/`
- `.pyc`
- 重複的名詞解釋與資料表文件

這些內容不是專案原始碼，加入壓縮檔只會增加體積與混亂。
